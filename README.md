# money note
money note is a very simple expense tracker made for personal use. Data is saved to localStorage.

## Installation
This is a simple Vue + vite app to install locally:

1. Clone this repo

```sh
git clone https://github.com/MaqLA/vue-money-tracker.git
```
2. Install modules
```sh
cd vue-money-note
npm i
```
3. Run app

```sh
npm run dev
    or
npx vite
```
